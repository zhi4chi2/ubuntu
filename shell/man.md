man 使用 less 作为 pager


使用示例
```bash
me@mypc:~$ man mkdir
me@mypc:~$
```

# Options
- -f - 相当于 `whatis`
- -k - 相当于 `apropos`
- -w - 显示 manual page 的地址
- -a - 显示所有相关的 manual page 。默认只显示第一个搜索到的 manual page 。例如 `man -a man` 将显示 man(1) 和 man(7)

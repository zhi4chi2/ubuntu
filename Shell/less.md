# 命令
- b/page up - 上面一页（看过的那一页）
- 空格键/page down - 下面一页（还没看过的那一页）
- 上箭头 - 上面一行
- 下箭头 - 下面一行
- G - 文件末尾
- g - 文件开头
- /characters - 向下搜索字符串
- n - 查找下一个，需要先使用 /characters 查找
- h - 显示帮助
- q - 退出

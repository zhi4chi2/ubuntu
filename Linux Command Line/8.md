# 编辑命令行

bash 使用 Readline 库实现命令行的编辑。

注意，下面的有些组合键可能被 GUI 识别为其它功能。

## 光标移动

- Ctrl-a - 移到行首
- Ctrl-e - 移到行尾
- Ctrl-f - 前移一个字符
- Ctrl-b - 后移一个字符
- Alt-f - 前移一个字
- Alt-b - 后移一个字
- Ctrl-l - 清屏


## 修改文本

- Ctrl-d - 删除光标处字符
- Ctrl-t - 使光标处字符与其前一个字符对调
- Alt-t - 使光标处字与其前一个字对调
- Alt-l - 从光标到字尾的字符都转为小写
- Alt-u - 从光标到字尾的字符都转为大写

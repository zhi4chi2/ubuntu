# 扩展

bash 在执行命令之前，对输入的命令行文本进行处理，这个过程叫做扩展(expansion)

```bash
me@mypc:~$ echo *
Desktop Documents Downloads examples.desktop GitBook Music Pictures Public Templates test Videos
me@mypc:~$ 
```
即 bash 将 * 扩展成了所有文件。


## 路径名扩展


计算隐藏文件夹的数量
```bash
me@mypc:~$ ls -d .* | wc -l
32
me@mypc:~$ ls -d .[!.]* | wc -l
30
me@mypc:~$ 
```
后一种方式去掉了 ., ..

## 波浪线扩展

```bash
me@mypc:~$ echo ~
/home/me
me@mypc:~$ echo ~eg
/home/eg
me@mypc:~$
```

## 算术扩展
```bash
me@mypc:~$ echo $((2+2))
4
me@mypc:~$ echo $(($((5**2))*3))
75
me@mypc:~$ echo $(((5**2)*3))
75
me@mypc:~$ echo $((5/2))
2
me@mypc:~$ 
```

算术扩展的格式是 $(()) 只支持整数运算，支持的运算符有
- +
- -
- *
- /
- %
- \*\* - 取幂

可以使用括号改变运算顺序。


```bash
me@mypc:~$ echo ~
/home/me
me@mypc:~$ echo ~eg
/home/eg
me@mypc:~$
```
